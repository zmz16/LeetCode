## C++

1. dp[i] ： 1到i为节点组成的二叉搜索树的个数为dp[i]
2. `dp[i] += dp[j - 1] * dp[i - j]`; ，j-1 为j为头结点左子树节点数量，i-j 为以j为头结点右子树节点数量
3. `dp[0] = 1`
4. 从左向右遍历
5. 举例

```cpp
class Solution {
public:
    int numTrees(int n) {
        vector<int> dp(n + 1);
        dp[0] = 1;
        for (int i = 1; i <= n; i++) {
            for (int j = 1; j <= i; j++) {
                dp[i] += dp[j - 1] * dp[i - j];
            }
        }
        return dp[n];
    }
};
```
