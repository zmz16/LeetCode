## 一维数组

和 [377组合总和Ⅳ](377组合总和Ⅳ.md) 基本一致，也是一个完全背包问题，不过这里的**物品是每次往上爬的阶数，背包是总台阶数n**。

```cpp
#include <iostream>
#include <vector>

using namespace std;

int main() {
    int n = 0, m = 0;
    cin >> n >> m;   // 种类，空间
    
    vector<int> dp(n + 1, 0);
    dp[0] = 1;

    for (int i = 0; i <= n; i++) {   // 台阶n
        for (int j = 1; j <= m; j++) {   // 每次爬的阶数m
            if (i - j >= 0) {
                dp[i] += dp[i - j];
            }
        }
    }
    cout << dp[n];
    return 0;
}
```
