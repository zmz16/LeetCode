## C++

1. `dp[i]`：分拆数字i，可以得到的最大乘积为`dp[i]`
2. `dp[i] = max({dp[i], (i - j) * j, dp[i - j] * j})`
3. dp[0]，dp[1]无意义，`dp[2] = 1`
4. 从前向后遍历
5. 举一个例子

```cpp
class Solution {
public:
    int integerBreak(int n) {
        vector<int> dp(n + 1);
        // 初始化
        dp[2] = 1;
        for (int i = 3; i <= n; i++) {
            for (int j = 1; j <= i / 2; j++) {
                dp[i] = max(dp[i], max(j * (i - j), j * dp[i - j]));
            }
        }
        return dp[n];
    }
};
```
