# 题目链接

https://leetcode.cn/problems/reverse-string-ii/description/

## 思路

根据题意，我们要反转的下标区间为：

`[0,k),[2k,3k),[4k,5k),⋯`

注意最后一个区间的右开端点要和 n 取最小值，防止下标越界。

## C++

```C++
class Solution {
public:
    string reverseStr(string s, int k) {
        int n = s.size();
        for (int i = 0; i < n; i += 2 * k) {
            reverse(s.begin() + i, s.begin() + min(i + k, n));
        }
        return s;
    }
};
```
